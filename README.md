# ITD42 Overlay Assistant

Кросс-браузерное расширение для Chrome и Firefox, которое показывает PNG-шаблон поверх `https://pixel.xn--d1ah4a.com/`.

## Что умеет

- Открывает панель управления на самой странице через popup.
- Загружает любой PNG-шаблон.
- Привязывает шаблон к координатам `X/Y` без масштабирования.
- Следит за pan и zoom у основной canvas-доски.
- Позволяет менять прозрачность шаблона.
- Позволяет включать и выключать отображение шаблона.
- Позволяет включать и выключать автовыбор цвета по клику на пиксель шаблона.
- Сохраняет настройки в `browser.storage.local`.

## Как работает автовыбор цвета

- PNG при загрузке конвертируется в ближайшие цвета палитры сайта.
- Если включён `Автовыбор цвета`, клик по непрозрачному пикселю шаблона пытается выбрать соответствующий цвет в правой палитре сайта.
- Автовыбор цвета можно в любой момент отключить в панели расширения.

## Установка в Chrome

1. Откройте `chrome://extensions/`.
2. Включите `Режим разработчика`.
3. Нажмите `Загрузить распакованное расширение`.
4. Выберите папку `itd42`.

## Установка в Firefox

1. Откройте `about:debugging#/runtime/this-firefox`.
2. Нажмите `Load Temporary Add-on`.
3. Выберите файл `manifest.json` внутри папки `itd42`.

## Использование

1. Откройте `https://pixel.xn--d1ah4a.com/`.
2. Откройте popup расширения.
3. Нажмите `Открыть панель`.
4. На странице появится панель, которую можно перетаскивать.
5. Загрузите PNG и задайте координаты `X` и `Y`.
6. При необходимости настройте прозрачность.
7. Используйте переключатели `Показывать шаблон` и `Автовыбор цвета`.
8. Если `Автовыбор цвета` включён, клик по пикселю шаблона попробует выбрать нужный цвет в палитре сайта.

## Сброс

Кнопка `Сбросить` очищает текущий шаблон и основные настройки, но оставляет саму панель доступной на странице.
